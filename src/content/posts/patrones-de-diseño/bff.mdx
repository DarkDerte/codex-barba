# Patrón BFF (Backend for Frontend)

Los patrones BFF (Backend for Frontend) son una solución arquitectónica que busca mejorar la comunicación entre el frontend y el backend de una aplicación. Este patrón se basa en la creación de un backend específico para cada tipo de cliente (web, móvil, etc.) o cada aplicación de frontend (por ejemplo al usar microfrontends), lo que permite optimizar la interacción y los datos que se envían a cada uno.
