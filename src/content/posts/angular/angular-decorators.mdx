# Decorators en Angular y TypeScript

Los decorators en Angular son una característica de TypeScript que permite añadir metadatos a clases, propiedades y métodos. Estos metadatos son utilizados por Angular para configurar y personalizar el comportamiento de los componentes, servicios y otros elementos de la aplicación.

## ¿Cómo funciona?

Los decorators son funciones que se aplican a una clase, propiedad o método utilizando el símbolo `@`. Por ejemplo, el decorator `@Component` se utiliza para definir un componente en Angular:

```typescript
import { Component } from "@angular/core";

@Component({
  selector: "app-root",
  templateUrl: "./app.component.html",
  styleUrls: ["./app.component.css"],
})
export class AppComponent {
  title = "mi-aplicacion-angular";
}
```
