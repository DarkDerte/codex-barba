# Pipes y custom pipes en Angular

Los pipes en Angular son una forma de transformar datos en las plantillas de tus componentes. Permiten formatear, filtrar o transformar datos de manera sencilla y reutilizable. Los pipes personalizados (custom pipes) te permiten crear transformaciones específicas que no están disponibles en los pipes integrados de Angular.

## ¿Cómo funciona?

Los pipes en Angular se aplican a los datos dentro de las plantillas utilizando el símbolo `|`. Por ejemplo, para formatear una fecha, puedes usar el pipe `date`:

```html
<p>{{ fecha | date:'dd/MM/yyyy' }}</p>
```

## ¿Cuándo usarlo?

Utiliza pipes cuando:

- Necesitas transformar datos en las plantillas de tus componentes.
- Quieres aplicar formateos o transformaciones comunes, como fechas, números o textos.
- Quieres crear transformaciones personalizadas que se reutilicen en diferentes partes de tu aplicación.
