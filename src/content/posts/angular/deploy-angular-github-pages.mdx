# Cómo desplegar tus aplicaciones Angular en GitHub Pages usando github-actions

En este vídeo, aprenderás a desplegar tus aplicaciones Angular en GitHub Pages utilizando GitHub Actions para automatizar el proceso.

## ¿Cómo funciona?

GitHub Actions es una herramienta de integración continua y entrega continua (CI/CD) que te permite automatizar tus flujos de trabajo directamente en tu repositorio de GitHub. Para desplegar una aplicación Angular en GitHub Pages, puedes crear un archivo de configuración de GitHub Actions que se encargue de construir tu aplicación y publicarla en la rama `gh-pages`.
