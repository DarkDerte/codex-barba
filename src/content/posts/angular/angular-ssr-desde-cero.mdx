# Angular SSR desde cero a despliegue

Angular SSR (Server-Side Rendering) permite renderizar aplicaciones Angular en el servidor, mejorando el rendimiento y la indexación por parte de los motores de búsqueda. En este artículo, aprenderás cómo implementar SSR en Angular desde cero hasta el despliegue.

## ¿Cómo funciona?

Angular utiliza un enfoque basado en módulos para implementar SSR. Esto significa que puedes crear un módulo específico para la renderización del lado del servidor y otro para la renderización del lado del cliente. Angular se encarga de gestionar la comunicación entre ambos módulos y de renderizar la aplicación en el servidor antes de enviarla al cliente.
