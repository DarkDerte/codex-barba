# Arrays, pilas y colas

Las estructuras de datos son fundamentales en la programación, y entender cómo funcionan los arrays, pilas y colas es esencial para cualquier desarrollador. Estas estructuras te permiten organizar y manipular datos de manera eficiente.

## Arrays

Los arrays son colecciones de elementos del mismo tipo (en la mayoría de los lenguajes, en lenguajes de tipado dinámico como JavaScript puedes tener arrays que contienen diferentes tipos de datos), accesibles mediante un índice. Son útiles para almacenar listas de datos y permiten acceder a los elementos de manera rápida.

## Pilas

Las pilas (stacks) son estructuras de datos que siguen el principio LIFO (Last In, First Out), lo que significa que el último elemento agregado es el primero en ser eliminado. Son útiles para implementar funcionalidades como deshacer acciones en aplicaciones o manejar llamadas a funciones.

## Colas

Las colas (queues) siguen el principio FIFO (First In, First Out), donde el primer elemento agregado es el primero en ser eliminado. Son útiles para gestionar tareas en orden de llegada, como en sistemas de impresión o procesamiento de solicitudes.
