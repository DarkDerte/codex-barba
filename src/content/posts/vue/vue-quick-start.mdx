# Requisitos

Tener instalado <a href="https://nodejs.org/es" target="_blank" rel="noopener noreferrer">Node.js</a> versión 18.3 o superior.

## Creación de una aplicación de Vue

En una terminal se ejecuta el siguiente comando

```bash
npm create vue@latest
```

## Nombramiento de la aplicación

Al ejecutar el comando le pedirá el que ingrese el nombre de la aplicación que realizará, recuerde que se creará una carpeta con el nombre que ingresó.

Nota: El nombre no puede llevar espacios, si quiere puede separar las palabras con un guión - Ejemplo: proyecto-inicial

## Configuración de la aplicación

Una vez ingrese el nombre del proyecto se le presentarán avisos para varias características opcionales como TypeScript y soporte de pruebas.

Para navegar por las opciones es con las flechas del teclado, para seleccionar una opción es con la tecla espacio.

## Arrancar la aplicación

Una vez que se creó la aplicación vamos a acceder a la carpeta del proyecto, instalar las dependencias y corren el proyecto en modo de desarrollo,

<a href="/screenshots/terminal/vue-quick-start/vue-create-project-finish.png" target="_blank" rel="noopener noreferrer">
  <img
    src="/screenshots/terminal/vue-quick-start/vue-create-project-finish.png"
    alt="Finalización del proyecto de Vue.js"
    width={800}
    height={300}
  />
</a>
<br />
<br />

Una vez en tu terminal estés dentro de la carpeta de tu proyecto, vamos a ejecutar el siguiente comando.

```bash
npm install
```

Una vez terminado el proceso de instalación de los paquetes necesarios vamos a ejecutar el siguiente comando para correr el proyecto en local en modo de desarrollo.

```bash 
npm run dev
```
