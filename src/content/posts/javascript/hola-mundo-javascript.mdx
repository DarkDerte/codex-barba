# Primeros pasos con JavaScript

Vamos a crear nuestro primer programa en JavaScript, que será un simple "Hola Mundo". Este es un ejercicio clásico para familiarizarse con la sintaxis del lenguaje.

Vamos a hacer una alerta en el navegador que muestre el mensaje "Hola Mundo".

Podemos crear un archivo HTML y agregar un script de JavaScript dentro de él. o podemos usar un editor HTML en línea como el <a href="https://www.w3schools.com/html/tryit.asp?filename=tryhtml_editor" target="_blank" rel="noopener noreferrer">Editor HTML gratis de w3schools</a>

## Código HTML de ejemplo en w3schools

```html
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>
```
## Agregando una alerta de JavaScript

Vamos a agregar un script de JavaScript dentro del archivo HTML para mostrar una alerta.

Este es el código que vamos a usar:

```javascript
<script>
  alert("Hola Mundo");
</script>
```

## Código HTML con nuestro script de JavaScript

```html
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

<script>
  alert("Hola Mundo");
</script>

</body>
</html>
```
